## 功能说明

### Tests

包含多种测试脚本：

- `weatherIndex.js` - 获取生活指数信息
- 其他测试脚本集

## 使用方式

```bash
node Tests/weatherIndex.js
```

### 环境配置

设置以下环境变量来启用通知功能：

- `LOCATION` - 城市代码（默认：101200101）
- `KEY` - 和风天气API密钥
